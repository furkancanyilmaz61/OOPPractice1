# 📚 Person Class and Example Objects

-------------------------------------------------------------------

## ✨ Project Features
-------------------------------------------------------------------
### 🛠️ Class Structure
- **Person Class**:
  - 🧑‍🎓 `Name`: The name of the person (string).
  - 🧑‍🏫 `Surname`: The surname of the person (string).
  - 🔢 `Yas`: The age of the person (int).
  - 📅 `BirthDate`: The birth date of the person (DateTime).
